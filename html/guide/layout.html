Container

통상 .navbar, .hero, .section, .footer 같은 곳에 .container 를 바로 아래에 위치

$desktop : maximum width 는 960px.
$widescreen : maximum width 는 1152px.
$fullhd : maximum width 는 1344px.
960, 1152, 1344 는 12와 16 둘다 나누기가 가능해서 선택됨
